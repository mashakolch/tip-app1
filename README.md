Описание: React-приложение для просмотра карточек животных с функционалом избранного и детальным просмотром.
Функционал:Просмотр карточек животных - сетка карточек с изображениями и описаниями.
Технические особенности
React Hooks: useState, useEffect для управления состоянием
Компонентная архитектура: Разделение на переиспользуемые компоненты
CSS-анимации: Плавные переходы и hover-эффекты
